# Android Runtime Template

这是 V2 开始引入的固定 Android Runtime 模板。

## 当前定位

这一版只提供最小工程骨架和接口位置，不追求完整能力实现。

它的目标是为后续版本提供：

- 固定 Android Studio 工程模板
- 固定 Runtime 包结构
- 固定任务执行模型
- 固定资源加载位置

## 这一版不保证

- 不保证已经可独立完成所有自动化能力
- 不保证已经可导出旧脚本
- 不保证已经和 Python 侧一一打通

## 后续版本会继续补齐

- Accessibility 点击/返回
- MediaProjection 截图
- 模板匹配
- OCR
- 资源导出链
- Kotlin 任务生成
